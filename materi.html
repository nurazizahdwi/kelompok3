Membuat List di HTML

HTML sudah menyediakan elemen untuk membuat list. Ada tiga macam jenis list yang bisa dibuat di HTML:


  1. Ordered List
  2. Unordered List
  3. Desciption List

1. Ordered List

Ordered List adalah Sebuah list yang beraturan

Ordered List menggunakan elemen 'ol' dan 'li'

2. Unordered List

Unordered List adalah Sebuah list yang tidak beraturan

Ordered List menggunakan elemen 'ul' dan 'li'

cara penggunaan :



ada beberapa type dalam ordered list yaitu: square,disc,none,circle

  • Square untuk kotak
  • Disc untuk lingkaran (terisi)
  • none untuk tanpa tanda
  • circle untuk lingkaran (tak berisi)

Cara penggunaannya cukup mudah yaitu dengan cara menambahkan ul type="square/disc/none/circle"

3. Description List

Description List adalah list yang berisi deksripsi atau penjelasan dari sesuatu.


Ada tiga tag yang digunakan untuk membuat description list:

  • (dl)(description list) tag untuk memulai description list
  • (dt) (description term) tag untuk membuat kata yang akan dideskripsikan
  • (dd) (description description) tag untuk membuat penjelasan dari kata.
